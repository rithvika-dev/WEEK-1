# JavaScript - Week 1

This folder contains my daily JavaScript learning progress and exercises for Week 1. 

## Structure

The directory is organized by day:

### `day-1(18-Feb-2026)`
Contains introductory JavaScript exercises:
- `max2.js`: Finds the largest of two numbers using `if...else`.
- `max3_ternary.js`: Finds the largest of three numbers using the ternary operator.
- `sum.js`: Calculates the sum of an array using a `for` loop.
- `min.js`: Finds the minimum element in an array using a `for` loop.
- `sum_fn.js`: A function to calculate the sum of an array.
- `search.js`: A function to search for an element in an array by value.
- `max3_fn.js`: A reusable function to find the largest of three numbers.

### `day-2(19-Feb-2026)`
Focuses on array and object manipulation in JavaScript.
- `emp_methods.js`: Practice working with an array of employee objects. Includes operations like:
  - Inserting new objects at specific indices using `splice()`.
  - Finding and removing items using `findIndex()` and `splice()`.
  - Finding and updating nested array values using `find()`.

## Running the Code

To run any of the files, you will need to have [Node.js](https://nodejs.org/) installed. 

Run a file from your terminal by navigating to the specific day's folder and executing:

```bash
node <filename>.js
```

For example:
```bash
cd "day-1(18-Feb-2026)"
node min.js
```
